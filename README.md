# File Organizer

## Overview

File Organizer is a Python automation project developed as part of the CodeAlpha Python Programming Internship. The script automatically detects and moves JPG image files from a source folder to a destination folder.

## Features

* Automatically detects JPG files
* Moves files from one folder to another
* Displays moved file names
* Shows total number of files moved
* Simple and beginner-friendly automation project

## Technologies Used

* Python 3.10
* os Module
* shutil Module

## Project Structure

CodeAlpha_File_Organizer

├── file_organizer.py

├── source_folder

├── destination_folder

└── README.md

## How to Run

1. Place JPG files inside source_folder.
2. Open terminal in the project folder.
3. Run:

python file_organizer.py

4. The files will be moved to destination_folder.
